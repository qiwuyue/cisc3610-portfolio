How To Install The App

  1. Open the web app in Chrome or Edge.
  2. Click the Install PWA button when it appears.
  3. Confirm the browser install prompt.
  4. Launch Instrument Explorer from your device like a regular app.

Web Version

Users can also run the app directly in the browser without installing it.
