Instrument Explorer

This page explains what the app does, how users install it, and how programmers can replace the JSON data with a new learning set.

User Guide

Features

  • Browse instrument topics from a dynamically generated menu.
  • Read short educational descriptions and fun facts.
  • View curated images for each instrument.
  • Play audio examples directly inside the app.
  • Install the app as a PWA and use cached content offline after the first visit.

Installation

How To Install

  1. Open index.html through the hosted website or Live Server.
  2. Use Chrome or Edge for the best PWA install support.
  3. Click the Install PWA button when it appears.
  4. Confirm the browser install prompt.
  5. Open the installed app from the desktop, start menu, or app launcher.

Programmer Documentation

How To Modify The Topic Dataset

The app reads its learning content from topics.json. To create a new learning set, replace the existing objects with new topic objects that follow the same structure. The app also includes custom-learning-set.json as a template for user-related information and custom topic data. 

{
  "id": 1,
  "title": "Piano",
  "category": "Keyboard",
  "text": "The piano is a keyboard instrument that produces sound when keys are pressed.",
  "image": "images/piano.png",
  "audio": "audio/piano.mp3",
  "fact": "A standard piano has 88 keys."
}

Required Fields

  • id: a unique number for the topic.
  • title: the name shown in the topic menu.
  • category: a short topic group or type.
  • text: the main educational paragraph.
  • image: the relative path to the topic image.
  • audio: the relative path to the topic audio file.
  • fact: a short extra fact displayed under the lesson.

Steps To Add A New Topic

  1. Add the image file to the images folder.
  2. Add the audio file to the audio folder.
  3. Add a new object to topics.json.
  4. Use lowercase file extensions, such as .png and .mp3.
  5. Add the new image and audio paths to sw.js so they are available offline.
